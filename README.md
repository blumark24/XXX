## 📋 وصف المشروع
Smart HSR هو نظام رقمي مخصص لإدارة الملاحظات الميدانية ومتابعة جودة المشاريع
باستخدام أدوات الذكاء الاصطناعي والرؤية الحاسوبية، ويعمل محليًا أو عبر Firebase بدون اشتراك.

## 🧠 المميزات
